\chapter{项目测试}

\section{前端测试}

根据UI效果图进行UI测试
\begin{enumerate}
\item 观察APP的用户界面(如菜单、对话框、窗口和其它可规控件)是否符合UI稿
\item 不同的连接页面之间导航链接是否有效，是否跳转是否正确
\item 输入框说明文字的内容与产品需求一致
\item 某页无数据时、断网时、有网但接口异常时的状态页是否和UI一致
\end{enumerate}

\section{接口测试}
\begin{enumerate}
\item Get,post发送和返回是否请求正常
\item 查看请求参数和返回参数结果是否正常
\end{enumerate}
（测试接口工具：postman）
